# Installation

## Prerequisites

- Python 3.10 or higher
- pip (Python package manager)
- git (recommended)

## Install with pip

The recommended way to install Romyq is via pip:

```bash
pip install romyq
```

## Verify installation

```bash
romyq --version
```

You should see output similar to:

```
Romyq v0.10.3
```

## Upgrade

To upgrade to the latest version:

```bash
pip install --upgrade romyq
```

## Uninstall

To remove Romyq:

```bash
pip uninstall romyq
```

This removes the CLI tool but preserves any project configurations.
